# kintone-basejs-sdk

> The SDK of kintone REST API client use for kintone-nodejs-sdk and kintone-js-sdk

## Install

- on NPM

```bashshell
$ cd your-project
$ npm install --save kintone-basejs-sdk
```

- On local

```bashshell

$ cd your-dev-folder
$ git clone https://github.com/kintone/kintone-basejs-sdk
$ cd kintone-basejs-sdk
$ npm link
```
